Vivado Installtion process

  1. Download Vivado HLx 2020.2: All OS installer Single-File Download (TAR/GZIP - 43.07 GB)

click on this link Vivado 2020.2

  1. After downloading go the folder where you have downloaded the tar.gz file and extract that.

  2. Start Screen Visible

  1. Click next

  1. Select vitis and click next

  1. Check checkboxes according to this image and click next.

  1. Check checkboxes and click next

  1. Change the loaction if needed.

  1. Click next and wait for Installtion.

  1. Download Board file using click on this link :- Pynq-z2 . Unzip the .zip file and copy that folder (named pynq-z2) to Location : C:\Xilinx\Vivado\2020.2\data\boards\board_files\ (Change location according to installtion path)
